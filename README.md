# Daily Dashboard Widgets

This project contains four school-appropriate widgets for a Google Site:

- Fun Fact of the Day
- Joke of the Day
- Word of the Day
- On This Day

Each widget changes automatically every day.

## Embed links after GitHub Pages is turned on

Replace YOUR-USERNAME with your GitHub username:

- https://YOUR-USERNAME.github.io/daily-dashboard-widgets/fun-fact/
- https://YOUR-USERNAME.github.io/daily-dashboard-widgets/joke/
- https://YOUR-USERNAME.github.io/daily-dashboard-widgets/word/
- https://YOUR-USERNAME.github.io/daily-dashboard-widgets/on-this-day/

## Editing the content

Facts are in `data/facts.json`.
Jokes are in `data/jokes.json`.
Words are in `data/words.json`.
On This Day entries are in `data/history.json`.
